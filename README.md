# 漢字クエスト ｜ 小4漢字の冒険

小学4年生の新出漢字（教科書 22〜62ページ・全53字）を、**冒険バトルRPG**で楽しくおぼえる学習アプリです。
iPad（Safari）での毎日の練習に特化しています。本番テスト **2026年6月17日** で100点を目指そう！

- 🌐 **完成版（フレームワーク不要・ビルド不要）**：HTML / CSS / JavaScript だけで動きます
- 📱 **iPad対応**：ホーム画面に追加すれば、アプリのように・オフラインでも遊べます（PWA）
- 💾 **進捗は端末に自動保存**（localStorage）。ログイン不要・サーバー不要

---

## 遊び方

1. **⚔️ ぼうけんに でる** … ステージ（章）を選んでバトル開始
2. **よみ問題**（自動採点）… 出てきた言葉の読みを4つの中からタップ。正解で敵に攻撃！
3. **かき問題**（自己採点）… ひらがなのヒントを見て、指やApple Pencilで漢字を手書き。
   - **👀 お手本** … 正しい字を表示
   - **🧽 けす** … 書き直し
   - 書けたら **✓ かけた！** ／ あやしいときは **もういちど**
4. **コンボ**を続けると **会心の一撃**！ 経験値がたまると **レベルアップ** ⭐
5. 章のボスを倒すと **次のステージが解放**。全10章クリアで **最終ボス「漢字王」** に挑戦！

### 毎日つづける工夫
- 🔥 **れんぞく日数**（ストリーク）… 毎日遊ぶとのびる
- 🎯 **せいかい率** と 📖 **おぼえた字（◯/53）** がひと目でわかる
- 🔁 **弱点とっくん** … まちがえた漢字だけを集中れんしゅう（努力の方法＝苦手を見つけて繰り返す）
- ⏳ **本番テストまでのカウントダウン** で、のこり日数がわかる

## iPadでアプリとして使う（ホーム画面に追加）

1. iPadの **Safari** で公開URLを開く
2. 画面下（または右上）の **共有ボタン** → **「ホーム画面に追加」**
3. ホームに出た **漢字クエスト** のアイコンから起動
   - 一度オンラインで開けば、その後は **オフラインでも** 遊べます

## GitHubで公開する手順（保護者向け・はじめてでもOK）

このフォルダ（`kanji-quest`）が、そのまま公開できる完成品です。

### A. GitHubにアップする
1. GitHubで **新しいリポジトリ** を作成（例：`kanji-quest`）。READMEなどは追加しなくてOK
2. このフォルダで、ターミナルから次を実行（`あなたのID` と `kanji-quest` は自分のものに置きかえ）：

   ```bash
   git remote add origin https://github.com/あなたのID/kanji-quest.git
   git branch -M main
   git push -u origin main
   ```

   ※このフォルダは **すでに git の準備（init・commit）済み** です。上の3行を実行するだけで公開できます。

### B. GitHub Pages を有効にする
1. GitHubのリポジトリ画面 → **Settings**（設定）→ 左メニュー **Pages**
2. **Build and deployment** の **Source** を **「Deploy from a branch」**
3. **Branch** を **`main`** ／ フォルダは **`/ (root)`** にして **Save**
4. 1〜2分待つと、上部に公開URLが表示されます：
   `https://あなたのID.github.io/kanji-quest/`
5. そのURLを **iPadのSafari** で開けば完成！（上の「ホーム画面に追加」へ）

> 更新したいときは、ファイルを直して `git add . && git commit -m "更新" && git push` すると、数分後に反映されます。

## 漢字データの直し方（必要なら）

漢字・読み・熟語のデータは **`js/kanji-data.js`** の1か所にまとまっています。

```js
"信": { page: 22, ch: 1, read: ["信号", "しんごう"], write: ["じしん", "自信"] },
//  字          ページ 章   読み問題[表示する語, 答えのよみ]  書き問題[ひらがなヒント, 答えの漢字]
```

- `read` … よみ問題に出す熟語と、その正しい読み（4択の正解になります）
- `write` … かき問題のひらがなヒントと、答えの漢字
- 章（ステージ）の構成は同ファイル下部の `CHAPTERS` で調整できます
- 直したら保存してリロード（GitHub公開版は push で反映）

## 正直な注意点（先にお伝えします）

- **手書きは自動採点していません。** 文字認識は不正確になりやすいため、「お手本と見くらべて自分で○×をつける」自己採点方式です（**読み問題は自動採点**です）。
- **オフライン動作は「初回にオンラインで開いたあと」** から有効になります（PWAの仕組み）。
- **お手本の字体はiPad標準のフォント**です。教科書体（とめ・はね・はらいが厳密な書写体）とは細部が異なる場合があります。
- 漢字53字・読み・熟語は教科書をもとに作成していますが、**最終的なテスト範囲は保護者の方が教科書でご確認**いただけると安心です。
